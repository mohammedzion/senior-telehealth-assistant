# Senior Telehealth Assistant

An AI-powered telehealth system designed to help senior citizens manage healthcare easily.

## Features
- Patient login authentication
- Appointment management
- Exercise monitoring
- Progress report tracking
- Voice assistant navigation

## Technology Stack

Frontend:
- React.js
- JavaScript
- CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## Project Structure

backend/ → Node.js API and database connection  
first-react/ → React frontend application

## Installation

Backend
cd backend  
npm install  
node server.js  

Frontend
cd first-react  
npm install  
npm start
